# Vietnam Market Breadth - Cơ hội

Interactive market breadth chart for top-100 Vietnam stocks (HOSE + HNX).

**Live chart:** https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/market_breadth.html

## Setup (one-time)

1. Create a free GitHub account at https://github.com
2. Create a new **public** repository (e.g. `vn-market-breadth`)
3. Upload this entire folder to the repo (drag-and-drop on GitHub)
4. Go to **Settings → Pages → Source** → set branch to `gh-pages`, folder `/`
5. Go to **Actions** tab → enable workflows if prompted

The chart auto-updates every weekday at 17:35 Vietnam time.  
You can also trigger it manually: **Actions → Update Market Breadth Chart → Run workflow**.

## Local use

Double-click `run_market_breadth.bat` to generate and open the chart locally.
